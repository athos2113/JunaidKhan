PROJECTS

Route Planning App

A mapping application with the ability to provide and visualize routes from one point to another in a map, by implementing Graphs and Graphing Algorithms.

Smart Text Editor App

A text editing app that incorporates “intelligent” behaviors of modern-day text interfaces including autocomplete, flagging misspelled words and spelling auto-correct.

Geospatial Visualization App

Interactive Visualization of a large dataset tagged by Geospatial Information. Built using a package called Unfolding Maps, which is a library for interactive maps and geovisualizations.

Social Network Analysis

Analysis of live social network data (Facebook) to discover Egonet and find connections within communities. Implemented using Graph and Graph Algorithms

Text Prediction App

A Shiny App that predicts the next word, using NLP. The predicted word is obtained from the n-grams matrices, comparing it with tokenized frequency of 2, 3 and 4 grams sequences.

MovieGuru

A Movie Recommender system that reads in live movie data and recommends Movies to a User based on their ratings of several movies

Creating Customer Segments

Analysis of customer's annual spending data. Applied PCA and K-means clustering to best describe the variation in different kinds of customers that a wholesale distributor interacts with. Doing so would equip the distributor with insight into how to best structure their delivery service to meet the needs of each customer.

Baby Names

Using data on the popularity of different baby names in the United States from the past several decades, we will be able to compare different names’ popularity over time. While the data we have collected for this project is from the United States, we can use data from other countries as well.

Spam Detection System

We will be using the Naive Bayes algorithm to create a model that can classify SMS messages as spam or not spam, based on the training we give to the model. This is a supervised, binary classification problem as messages are classified as either 'Spam' or 'Not Spam' and nothing else.
